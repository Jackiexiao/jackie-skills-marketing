# jackie-skills-marketing

一键装好热门营销增长技能（懒人版）。
按场景直接用，快速拿结果。

## 一键安装

```bash
npx skills add Jackiexiao/jackie-skills-marketing --all
```

> 这条命令适用于几乎所有主流 AI 编程工具（Claude Code / Codex / Trae / OpenClaw 等）。

## 这套适合谁

- 独立开发者做增长
- 产品/运营/内容团队要提转化

## 包含技能（14）+ 直接可用示例

- `audit-website`：网站全量体检（性能/SEO/可用性）
  - 示例：`审计我的官网并按影响优先级给出 TOP10 修复项。`
- `seo-audit`：SEO 问题排查
  - 示例：`为什么我的 landing page 不排名？做一份 SEO 诊断。`
- `programmatic-seo`：规模化 SEO 页面策略
  - 示例：`为“城市+行业”落地页设计可批量生成模板。`
- `copywriting`：转化文案撰写
  - 示例：`给我的 AI 记账产品写首页首屏文案和 CTA。`
- `copy-editing`：文案润色提效
  - 示例：`把这段文案改得更短、更有行动力。`
- `content-strategy`：内容方向与发布节奏
  - 示例：`给我做一个 30 天内容选题日历，目标是获客。`
- `marketing-ideas`：快速产出增长点子
  - 示例：`给我 20 个低预算 SaaS 增长动作，按优先级排序。`
- `marketing-psychology`：心理学驱动转化
  - 示例：`重写价格页，加入损失厌恶和社会认同机制。`
- `product-marketing-context`：先统一定位再做传播
  - 示例：`帮我整理产品定位文档：人群、痛点、差异化、证据。`
- `pricing-strategy`：定价与套餐设计
  - 示例：`给我设计 3 档 SaaS 套餐，并说明每档逻辑。`
- `page-cro`：页面转化优化
  - 示例：`分析这个注册页，给出 5 个可 A/B 测试改动。`
- `launch-strategy`：新功能/新产品发布节奏
  - 示例：`为“AI 自动客服”做 14 天上线节奏和渠道分发计划。`
- `analytics-tracking`：埋点与归因
  - 示例：`给我做 GA4 事件追踪方案，覆盖注册到付费漏斗。`
- `schema-markup`：结构化数据提升搜索展示
  - 示例：`为这个产品页生成 JSON-LD schema 并解释字段。`

## 快速用起来（3 步）

1. 一条命令安装
2. 挑一个最痛点场景（SEO / 文案 / 转化）直接跑示例
3. 先做小改动验证，再做大改版
